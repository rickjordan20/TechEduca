Projeto TechEduca

O TechEduca é um projeto desenvolvido pelos alunos do curso Técnico em Desenvolvimento de Sistemas do Senac DF, com o objetivo de aplicar, na prática, os conhecimentos de desenvolvimento web.

O projeto consiste na criação de uma plataforma educacional, utilizando tecnologias como HTML, CSS e JavaScript, integrando conceitos de front-end, manipulação do DOM e consumo de APIs.

Durante o desenvolvimento, os alunos trabalharam com versionamento de código no GitHub, organização de projetos, boas práticas de programação e construção de interfaces interativas, preparando-se para desafios reais do mercado de tecnologia.
